# dotfiles

Dotfiles for:
- `bat`
- `btop`
- `ghostty`
- `git`
- `starship`
- `tmux`
- `zsh`

```
dotfiles
├── bat
│   └── config
├── btop
│   └── btop.conf
├── ghostty
│   ├── config
│   └── gruvbox-dark-hard.icns
├── git
│   └── .gitconfig
├── README.md
├── starship
│   └── starship.toml
├── tmux
│   ├── gruvbox-dark.conf
│   └── tmux.conf
└── zsh
    ├── .zprofile
    └── .zshrc
```
